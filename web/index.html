About this Project:

This simple website was hosted on Amazon Simple Storage Service - S3 bucket. By utilizing Terraform, I was able to define the AWS resources required for
the S3 bucket, including its configuration, policies, and permissions. GitHub Actions was set up to respond to changes pushed to the main branch of the repository,
automatically triggering the redeployment process, ensuring seamless updates to the website.

Summary of the resources used:

  • - Terraform for defining and managing AWS resources.
  • - GitHub Actions for automation and CI/CD processes.
  • - S3 bucket serving as the hosting platform for this website.
  • - Route53 for custom domain registration and usage.
  • - Cloudfront Distribution to cache content to edge location and improve website performance.
