About this Project:

This website is hosted on Amazon Simple Storage Service (S3). By utilizing Terraform, I was able to build the Infratructure as code (IaC).
This includes deploying all the AWS resources needed to host the website and have it running in a secure manner with low latency.
GitHub Actions was configured to respond to changes to changed in the code base in either Dev or Prod environment which triggers
redeployment process and ensuring seamless updates to the website.

Summary:

  • - Terraform for deploying and managing AWS resources as code.
  • - GitHub Actions for automation and CI/CD processes.
  • - S3 bucket serving as the hosting platform for this website.
  • - Route53 for custom domain registration and usage.
  • - Cloudfront Distribution to cache content to edge location and improve website performance.
